# Project Blueprint

## Overview

This project is a visually appealing and interactive lottery number generator web application. It allows users to generate a set of 6 unique random numbers with a single click.

## Style, Design, and Features

*   **UI:** A clean, modern interface with a clear heading, a dedicated area for displaying the lottery numbers, and a prominent "Generate Numbers" button.
*   **Functionality:** Clicking the button instantly generates 6 unique random numbers between 1 and 45.
*   **Styling:** 
    *   **Layout:** Centered, responsive layout that works on all screen sizes.
    *   **Color Palette:** A vibrant and energetic color scheme to make the experience engaging. Each number ball will have a different color.
    *   **Typography:** Expressive and clear fonts to enhance readability.
    *   **Effects:** Subtle shadows and transitions to create a sense of depth and interactivity.
    *   **Background:** A gentle noise texture on the background for a premium feel.
